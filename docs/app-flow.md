# App Flow

JongSart currently demonstrates a local booking flow:

1. Customer opens the Flutter app.
2. Customer browses clinics, doctors, treatments, promotions, or skin concerns.
3. Customer submits an appointment request.
4. The request starts as pending confirmation.
5. Clinic staff manage the request in the demo staff screen.
6. Staff can confirm, reschedule, cancel, or complete the booking.
7. Customer can leave a review after a completed visit.

The current version uses mock/local data and `SharedPreferences`.
